# 重定向

* cat － 连接文件
* sort － 排序文本行
* uniq － 报道或省略重复行
* grep － 打印匹配行
* wc － 打印文件中换行符，字，和字节个数
* head － 输出文件第一部分
* tail - 输出文件最后一部分
* tee - 从标准输入读取数据，并同时写到标准输出和文件

I/O 重定向允许我们更改输出地点和输入来源。一般地，输出送到屏幕，输入来自键盘， 但是通过 I/O 重定向，我们可以做出改变。

### 重定向标准输出

我们使用 “>” 重定向符后接文件名来重定向**标准输出**到除屏幕 以外的另一个文件。

    [me@linuxbox ~]$ ls -l /usr/bin > ls-output.txt
