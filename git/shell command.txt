
关于目录:

.		当前目录
..		上一层目录
./a/b	当前目录下的a目录下的b目录
~		用户主目录
/		根目录


路径相关命令:

pwd			显示当前目录
cd ..		切换到上一层目录
cd ../..	上上层目录
cd 	~		切换到用户主目录
cd -		切换到上次所在的目录
cd /		根目录

ls -al		显示当前目录下所有文件(包括隐藏文件)
ls -l		显示当前目录下文件（不包括隐藏文件）
ll			等同于ls -l

mkdir a		在当前路径下创建目录a
touch b		在当前路径下创建文件b

cp a b		拷贝文件a到b
mv a b		移动文件a到b
